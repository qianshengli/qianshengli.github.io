Endpoints

Integrate transaction grids, widget listings, and state telemetry with the Antigravity developer REST API endpoints.

GET /v1/projects

Retrieve a list of active canvas projects in the authorized workspace.

Query Parameters

Parameter Type Description
limit optional integer The number of items to return. Max limit is 100. Default is 20.
status optional string Filter projects by status (e.g. active, archived).
POST /v1/projects

Create a new canvas project with custom layout structures and initial style tokens.

Request Body Parameters

Parameter Type Description
name required string The display name of the workspace project canvas.
template optional string Blueprint template id (e.g. landing, saas, mobile).
DELETE /v1/projects/:id

Permanently delete a project from the workspace. This action is irreversible.

Path Parameters

Parameter Type Description
id required string The unique identifier token of the targeted project.
