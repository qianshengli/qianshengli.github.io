Installation

Get up and running with the Antigravity package CLI compiler in your project directory.

Prerequisites

Before installing, ensure that your local development environment meets the following requirements:

  • Node.js: Version 18.0.0 or higher is required.
  • Package Manager: Node Package Manager (npm), Yarn, or pnpm.
  • Framework: Vite, Next.js, or any environment supporting CSS-in-JS or global stylesheets.
Workspace Setup If you are running in a monorepo workspace, make sure to execute commands inside the designated project subdirectory.

Install Packages

Run the install command using your package manager of choice to add the core compiler package:

# Install Antigravity compiler as dev dependency
npm install -D @antigravity/compiler
# Add as dev dependency via yarn
yarn add -D @antigravity/compiler
# Install dev dependency using pnpm
pnpm add -D @antigravity/compiler

Tailwind Integration

If you're using Tailwind CSS, make sure to configure your postcss plugins properly. The PostCSS plugin has moved to a separate package in recent versions.

PostCSS Tailwind Warning Do not use tailwindcss directly as a PostCSS plugin. To continue using Tailwind CSS with PostCSS, install @tailwindcss/postcss and update your PostCSS config file.
